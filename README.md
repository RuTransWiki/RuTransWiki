# RuTransWiki

Сборник материалов по трансгендерному переходу для русскоязычных транс-персон.

## Как внести правки

### 1. Создайте аккаунт на GitHub и установите git

Зарегистрируйтесь на [github.com](https://github.com). Затем установите git:

- **Windows:** скачайте и установите [Git for Windows](https://git-scm.com/download/win)
- **macOS:** откройте Терминал и выполните `git --version` — если git не установлен, система предложит его установить
- **Linux:** установите через пакетный менеджер, например `sudo apt install git`

После установки представьтесь git (один раз):

```
git config --global user.name "Ваше имя"
git config --global user.email "ваш@email.com"
```

### 2. Сделайте форк репозитория

Откройте [github.com/RuTransWiki/RuTransWiki](https://github.com/RuTransWiki/RuTransWiki) и нажмите кнопку **Fork** в правом верхнем углу. Это создаст копию репозитория в вашем аккаунте.

### 3. Склонируйте форк на свой компьютер

Откройте ваш форк на GitHub (он будет по адресу `github.com/ВашеИмя/RuTransWiki`), нажмите кнопку **Code** и скопируйте ссылку. Затем выполните в терминале:

```
git clone https://github.com/ВашеИмя/RuTransWiki.git
```

Это скачает репозиторий в папку `RuTransWiki` на вашем компьютере.

### 4. Внесите изменения

Откройте папку `RuTransWiki` как хранилище в [Obsidian](https://obsidian.md) и вносите правки в обычном режиме. После редактирования зафиксируйте изменения:

```
git add .
git commit -m "Краткое описание изменений"
```

### 5. Отправьте изменения в ваш форк

```
git push
```

### 6. Создайте Pull Request

Откройте ваш форк на GitHub. Появится кнопка **Compare & pull request** — нажмите её, опишите внесённые изменения и нажмите **Create pull request**. После проверки правки будут добавлены в основной репозиторий.
