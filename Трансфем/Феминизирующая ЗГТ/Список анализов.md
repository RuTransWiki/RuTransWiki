
| Анализ                | Целевые значения                                                          |
| --------------------- | ------------------------------------------------------------------------- |
| Эстрадиол (E2)*       | 100–200 pg/mL (367–734 pmol/L)[^wpath_levels]                             |
| Тестостерон (T) общий | <50 ng/dL (<1.73 nmol/L)[^wpath_levels], в идеале — <30 ng/dL (<1 nmol/L) |

*\* При [[Феминизирующая ЗГТ#Монотерапия|монотерапии]] могут быть необходимы уровни эстрадиола, превышающие 200 pg/mL, обычно — 200–300 pg/mL (734–1101 pmol/L)*

## Единицы измерения

При публикации вопросов на счёт гормональной терапии, всегда указывайте результаты анализов **вместе с единицами измерения**.

Для конвертации между различными единицами измерения используйте следующие конвертеры:

- Эстрадиол: https://unitslab.com/node/113
- Тестостерон: https://unitslab.com/node/136
